# Tauri Icons

This directory should contain the following icon files for the Tauri application:

- `32x32.png` - 32x32 pixel PNG icon
- `128x128.png` - 128x128 pixel PNG icon  
- `128x128@2x.png` - 256x256 pixel PNG icon (for high DPI displays)
- `icon.icns` - macOS icon file
- `icon.ico` - Windows icon file

## Creating Icons

You can create these icons from a single high-resolution source image (recommended 512x512 or 1024x1024) using tools like:

- **Online**: https://iconverticons.com/online/
- **macOS**: Use `iconutil` command
- **Windows**: Use online ICO converters
- **Linux**: Use `convert` from ImageMagick

## Icon Design

The icon should represent the Drawing Inspiration app - consider using:
- Art/drawing related symbols (paintbrush, palette, canvas)
- Creative/artistic elements
- The app's color scheme (dark theme with blue accents)

For now, you can use placeholder icons or create simple ones using any image editing software.
